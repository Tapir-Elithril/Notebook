## 授课教师
赵俊博

### 认定
专业必修课

### 课程教材
西瓜书  

### 分数构成
- attendance 15% （两次提前通知的quiz）  
- homework 20% （两次各10%，包括理论与编程）  
- project 25% （代码与报告各占40%，kaggle上的正确率按0.5,0.6,...,0.9分分档占剩余20%）  
- final-exam 40% (选择10 * 2% + 8个大题)  

### 学习建议
赵俊博上课的内容较赵洲更贴近前沿，会重点讲授深度学习和强化学习，还会有实验课带着入一下fine-tuning。  
点名是一学期两次提前通知的quiz，题目可以参考历年智云课堂，有一次人数过少突击点名，在老师手机上打名字，并声称以后会随机点名，之后到课人数回升就没再点过名了，没有明确的证据表明突击点名对成绩有影响。  
project仍然是kaggle上的[叶子分类](https://www.kaggle.com/competitions/classify-leaves)，单人完成，比较简单，本地训练比较费时，建议尽早开始。  
期末考试内容参考复习课，重点难点主要在极大似然估计，期末考试较简单。
