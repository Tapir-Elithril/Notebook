- Default Ctor  
- Ctor take two integers as parameters
- Copy ctor  
- Arithmetic operations:`+ - * /`  
- Relational operators:`< <= == != >= >`  
- Typecast to double  
- To string  
- Inserter and extractor for streams  
- Conversion from a finite decimal string like:`1.414`
